I am trying to make the ultimate guide of AI/ML/Data science related position interviews. Here are some questions that I have been asked before, categorized into groups. Of course that there are some elemets missing. Please identify the missing elements and ask me if I want to add them into the list. I am planning to make this guide into a simple app that checks your readiness and weakness (maybe on web) later.

0. Self introduction
	- background / career history
	- strengths & weaknesses
	- reason for job change
	- short-term vs long-term career goals
	- "tell me about yourself" vs formal 自己紹介
1. 応募動機
	- Why this company
	- Why this role specifically
	- Why now (timing)
	- What value can you provide this company
	- What can the company provide you
2. Technical quiz
	- coding
		- competitive 
			- whiteboard
			- ide
			- data structures & algorithms
			- debugging / code review
		- tool use
			- SQL
			- REST calls
			- Python / pandas
			- shell / git
			- notebook workflow
		- system design
			- backend
			- special purpose
			- ML system design (recommender, search ranker, etc.)
			- data pipeline / ETL
			- MLOps / model serving / monitoring
	- quizzes
		- Machine learning
			- Precision / Recall
			- overfitting
			- model structures
			- bias / variance tradeoff
			- regularization (L1/L2, dropout)
			- cross-validation
			- feature engineering
			- metrics (F1, ROC-AUC, PR-AUC, RMSE/MAE, calibration)
			- hyperparameter tuning
		- Deep learning
			- CNN
			- RNN
			- Transformer / attention
			- optimizers
		- Statistics & probability
		- Linear algebra / calculus / optimization
		- A/B testing & experimentation
		- NLP fundamentals
		- Computer vision fundamentals
		- Recommendation systems
		- Reinforcement learning
		- Time series
		- Responsible AI / fairness / safety
		- LLM
			- tokenization & embeddings
			- prompt engineering
			- RAG
			- fine-tuning (SFT, LoRA, PEFT, RLHF, DPO)
			- evaluation (benchmarks, LLM-as-judge)
			- agents & tool use
			- context window / caching / cost tradeoffs
		- Data analysis
3. Assignment
	- homework
	- homework discussion
	- take-home coding challenge
	- live pair programming / live coding
	- case study / business problem
	- presentation / demo of prior work
	- paper reading / research discussion
4. Past experience
	- customer facing
	- AI agent
	- finetuning LLM/VLM
	- production deployment / scale
	- team leadership / mentoring
	- cross-functional collaboration (PM, design, business)
	- research / publications / open source
	- data science projects (non-LLM)
	- incident response / on-call
	- performance optimization
5. STAR questions
	- act proactively
	- different opinions
	- push yourself over limits
	- handed instructions not clear
	- made a mistake
	- conflict resolution with teammates
	- tight deadlines / prioritization under pressure
	- failed project / lessons learned
	- giving & receiving feedback
	- leading without authority / influence
	- handling ambiguity
	- ethical dilemma (AI ethics)
	- dealing with technical debt
6. Alignment
	- Top 3 priorities
	- Expected Salary
	- Other applications
	- level of devote
	- remote/in office
	- career path
	- visa / work authorization
	- start date / notice period
	- benefits expectations
	- management track vs IC track
	- team size / structure preference
	- travel willingness

7. Questions about the company
	- team structure & roadmap
	- tech stack & tooling
	- growth / promotion path
	- evaluation criteria
	- onboarding
	- current challenges the team is facing

8. Screening / recruiter round
	- elevator pitch
	- expected-comp phrasing
	- logistics (timeline, process steps)

9. Post-interview
	- thank-you notes
	- follow-up cadence
	- negotiation
